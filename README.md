## About The App

E-Commerce Fitness Training and Nutrition Professional Website. Allows users to browse available packages for purchase.
Currently working on: registration, package details and view-to-buy process.

## Getting Started

To get a local copy up and running follow these simple steps after downloading the files from the Github.

### Prerequisites

To run this SPA, install it locally using npm and run mock API on a separate terminal:

```
$ npm install
$ npm run jsondb
$ npm start
```
